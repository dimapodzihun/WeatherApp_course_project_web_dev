# 🌦 Курсовий проєкт 2 курс, 2 семестр
## 📚 Дисципліна: Технології веб-розробки

## 📌 Тема
Розробка вебдодатку для аналізу та прогнозування погодних умов  
на основі даних з супутників та метеорологічних станцій.

---

## 🧩 Постановка задачі
Розробити вебсистему для:
- збору метеорологічних даних
- аналізу погодних умов
- прогнозування змін погоди

---

## 🚀 Можливості системи
- 🌍 Отримання погодних даних у реальному часі
- 📈 Короткострокове та довгострокове прогнозування
- 📊 Візуалізація кліматичних змін
- 🕓 Аналіз історичних погодних трендів
- ⚡ Інтерактивні графіки

---

## 🛠 Технології
- 🐍 Python (Flask)
- 🐘 PostgreSQL
- 🌐 HTML
- 🎨 Tailwind CSS
- ☁️ OpenWeather API
- 📊 Pandas
- ⚡ HTMX
- 📉 Python (Plotly)
